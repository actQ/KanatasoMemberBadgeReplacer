# Kanata Member Badge Replacer

**バージョン: v0.1.0**

YouTube のライブチャットで、**天音かなたチャンネル**のメンバーバッジ画像が表示されなくなっている問題を修正する Chrome 拡張機能です。  
非公式ファンWiki に掲載されている本来のバッジ画像に自動で差し替えます。

---

## 動作環境

- OS: Windows 11
- ブラウザ: Google Chrome

---

## インストール方法

### 手順1 — ZIPファイルをダウンロードする

このページ右上の「**Releases**」から最新版の ZIP ファイル（例: `kanata-member-badge-replacer-v0.1.0.zip`）をダウンロードします。

### 手順2 — ZIPを展開する

ダウンロードした ZIP ファイルを右クリック →「**すべて展開**」で、わかりやすい場所（デスクトップなど）に展開します。  
展開すると `extension` というフォルダができます。

### 手順3 — Chromeの拡張機能画面を開く

Chrome のアドレスバーに以下を入力して Enter を押します。

```
chrome://extensions
```

### 手順4 — デベロッパーモードを有効にする

画面右上に「**デベロッパー モード**」というトグルスイッチがあります。これを **オン** にします。

> ⚠️ オンにすると画面上部にボタンが3つ追加されます。これは正常です。

### 手順5 — 拡張機能を読み込む

「**パッケージ化されていない拡張機能を読み込む**」ボタンをクリックし、手順2で展開した **`extension` フォルダ** を選択します。

### 手順6 — 確認

一覧に「**Kanata Member Badge Replacer**」が表示されれば完了です。

---

## 使い方

インストール後、天音かなたその動画ページを開くと、ライブチャット内のメンバーバッジが自動で差し替わります。

---

## アップデート方法

1. 新しい ZIP をダウンロードして展開する（古いフォルダを上書きするか、別の場所に展開）
2. `chrome://extensions` を開く
3. 「Kanata Member Badge Replacer」の **更新** ボタン（🔄 アイコン）をクリック

## 注意事項

- この拡張機能は非公式のファンメイドツールです。天音かなた・ホロライブプロダクションとは無関係です。
- Chrome Web Store には公開していません。インストールには上記の手動手順が必要です。
- YouTube の仕様変更により動作しなくなる場合があります。
- 次の段階で、メンバー経過月数ごとの個別画像や、表示できないときの代替スタンプも追加できます。
- 何かしらの要因でバッジ表示が復活した場合、この拡張機能がバッジを上書きしてしまうため、この拡張機能は無効化&削除してください。

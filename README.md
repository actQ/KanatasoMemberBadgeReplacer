# ★THE かなた★ Badge Replacer

**バージョン: v0.1.0**

YouTube のライブチャットで、**天音かなたチャンネル**のメンバーバッジ画像を差し替える Chrome 拡張機能です。  

---

## 動作環境

- OS: Windows 11
- ブラウザ: Google Chrome

---

## インストール方法

### 手順1 — ZIPファイルをダウンロードする

以下のリンクから最新版の ZIP ファイルをダウンロードします。

**[ kanata-member-badge-replacer-v0.1.0.zip ](https://github.com/actQ/KanatasoMemberBadgeReplacer/releases/download/v0.1.0/kanata-member-badge-replacer-v0.1.0.zip)** をダウンロード

### 手順2 — Chromeの拡張機能画面を開く

Chrome のアドレスバーに以下を入力して Enter を押します。

```
chrome://extensions
```

### 手順3 — デベロッパーモードを有効にする

画面右上に「**デベロッパー モード**」というトグルスイッチがあります。これを **オン** にします。

### 手順4 — ZIPをドラッグ＆ドロップする

ダウンロードした **ZIP ファイル**を `chrome://extensions` の画面に**ドラッグ＆ドロップ**します。

### 手順5 — 確認

一覧に「**★THE かなた★ Badge Replacer**」が表示されれば完了です。

---

## 使い方

インストール後、天音かなたその動画ページを開くと、ライブチャット内のメンバーバッジが自動で差し替わります。

---

## アップデート方法

1. 古い拡張機能を `chrome://extensions` から削除する
2. 新しい ZIP をダウンロードして、同じ手順でドラッグ＆ドロップする

## 注意事項

- この拡張機能は非公式のファンメイドツールです。天音かなた・ホロライブプロダクションとは無関係です。
- Chrome Web Store には公開していません。インストールには上記の手動手順が必要です。
- YouTube の仕様変更により動作しなくなる場合があります。
- 何かしらの要因でバッジ表示が復活した場合、この拡張機能がバッジを上書きしてしまうため、この拡張機能は無効化&削除してください。
